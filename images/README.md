# Atlas Upholstering — Images

Place production images in this directory.

## Required Images

| Filename | Purpose | Recommended Size |
|----------|---------|------------------|
| `og-image.jpg` | Open Graph / social share image | 1200 × 630 px |
| `hero-bg.jpg` | Homepage hero background | 1920 × 1080 px |
| `workshop.jpg` | About page — workshop interior | 800 × 800 px |
| `repair-service.jpg` | Services page — repair section | 800 × 600 px |
| `reupholstering-service.jpg` | Services page — reupholstering section | 800 × 600 px |
| `custom-service.jpg` | Services page — custom manufacturing section | 800 × 600 px |
| `blog-upholstery-guide.jpg` | Blog post hero image | 1200 × 675 px |

## Alt Text Guidelines

All images must include descriptive, keyword-rich alt text. Examples:
- "furniture reupholstering Montreal — craftsman working on sofa fabric"
- "custom sofa manufacturing Montreal — Atlas Upholstering workshop"
- "upholstery fabric selection Montreal — velvet and linen samples"
- "furniture repair Montreal — before and after armchair restoration"

## Format Recommendations
- Use WebP format where possible for performance (with JPEG fallback)
- Compress all images (target: under 200KB for most, under 500KB for hero)
- Use `loading="lazy"` on all below-fold images (already set in HTML)
